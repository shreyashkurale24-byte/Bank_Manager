🔌 REST API

Every table is available as a JSON API under /api/v1/, authenticated with an API key sent as an X-API-Key header. Configure the key via the API_KEY environment variable — see .env.example.

curl http://localhost:5000/api/v1/tables \
  -H "X-API-Key: your-api-key"

The API is configured via environment variables, not from this page — see the project README for the full endpoint list.
